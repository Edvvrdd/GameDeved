Most major websites have a search query these days and to access them you need to be on their website.
While that is the beneficial to the website, it might not be the most beneficial to the user experience as the website usually display a lot of content that are meant to distract you. 
The goal of this app is to make a simple interface for the user to search for items without and distraction. 
another goal of the app is to show the hidden search syntaxs that most websites omit in order to achieve a more minimalistic aesthetic.
